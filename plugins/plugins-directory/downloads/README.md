# WPOpenData - Plugins Directory Downloads

Downloads from the Plugins Directory (data available from 2007 to 2017).

## Formats

- [CSV](plugin-directory-downloads.csv)

## Fields

- `date`: (date) Sampling date.
- `downloads`: (int) WordPress trend.
